#Examen de Robótica - ROS 2

Este repositorio contiene el desarrollo de ejercicios relacionados con simulación de robots en ROS 2, incluyendo cinemática directa e inversa.

## Estructura

- `ejercicio1/` → Cinemática Robot A
- `ejercicio2/` → Cinemática Robot bioinspirado
- `pub_nodos/` → Comunicación con tópicos y nodos

Cada ejercicio cuenta con su propio README con instrucciones específicas.
